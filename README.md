# 🌐 Automated AWS Static Website Hosting with Terraform

This project demonstrates how to fully automate the deployment of a modern, responsive static website on **Amazon Web Services (AWS)** using **Infrastructure as Code (IaC)** with Terraform.

## 🚀 Project Highlights
* **Zero Manual Config:** The entire infrastructure (S3 Bucket, Policies, and Website Hosting) was provisioned via code.
* **Modern Frontend:** A professional Cloud & DevOps Portfolio dashboard built with **Tailwind CSS**.
* **Public Access Management:** Configured AWS S3 bucket policies for secure yet public web access.

## 🛠️ Tech Stack
* **Cloud Provider:** AWS (S3)
* **IaC Tool:** Terraform
* **Frontend:** HTML5, Tailwind CSS

## 📂 Architecture
The project follows a modular Terraform structure:
- `main.tf`: Defines S3 bucket and access policies.
- `variables.tf`: Manages environment variables.
- `outputs.tf`: Outputs the live website URL.
